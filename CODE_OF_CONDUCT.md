# Code of Conduct

This project follows version 2.1 of the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be patient, assume good intent, and help
keep this a welcoming space for everyone — regardless of experience
level, background, or the shape of their homelab.

## Reporting

If you see behavior that violates the Code of Conduct, please email
**support@vetroscope.com**. Reports are handled confidentially by the
project maintainers.
